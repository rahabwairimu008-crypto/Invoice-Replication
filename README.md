#  Invoice Project

##  Overview
This project is a **Java-based Invoice System** designed to produce professional, formatted invoices.  
It was created as part of an academic assignment with strict rules (e.g., no arrays, no loops) and aims to replicate a real-world business invoice with pixel-perfect alignment.

##  Features
- Input customer details (Name, Address, Contact)
- Input product details (Item name, Quantity, Price)
- Automatic calculation of:
  - Subtotal
  - Taxes (if applicable)
  - Grand Total
- Clean, console-based output styled to resemble a professional invoice
- Multi-product support with precise alignment

##  Technologies Used
- **Language:** Java (console-based, no arrays/loops)  
- **IDE:** IntelliJ IDEA  
- **Version Control:** Git & GitHub
